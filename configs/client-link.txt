# Project Janus — VLESS Client Link Templates
# =============================================

# TEMPLATE: For restricted networks with SNI filtering (recommended)
# Uses Cloudflare Worker for SNI fronting
# Replace: YOUR_UUID, YOUR_ACCOUNT
vless://YOUR_UUID@janus-relay.YOUR_ACCOUNT.workers.dev:443?type=ws&encryption=none&path=%2Fsecretpath&host=janus-relay.YOUR_ACCOUNT.workers.dev&security=tls&sni=janus-relay.YOUR_ACCOUNT.workers.dev#janus-aules

# TEMPLATE: For networks without SNI filtering
# Uses Cloudflare IP directly (bypasses DNS blocking only)
# Replace: YOUR_UUID, CLOUDFLARE_IP, yourdomain.com
vless://YOUR_UUID@CLOUDFLARE_IP:443?type=ws&encryption=none&path=%2Fsecretpath&host=janus.yourdomain.com&security=tls&sni=janus.yourdomain.com#janus-cloudflare

# TEMPLATE: For home network testing (direct LAN)
# Replace: YOUR_UUID
vless://YOUR_UUID@192.168.0.103:8080?type=ws&encryption=none&path=%2Fsecretpath&host=janus.yourdomain.com&security=none#janus-local

# =============================================
# Which template to use:
#
#   1. Try "restricted networks" first (always works)
#   2. If no SNI filtering, "without SNI filtering" has less latency
#   3. "home network" is for local testing only
#
# Find your Cloudflare IP:
#   dig janus.yourdomain.com +short
#
# Generate a UUID:
#   /usr/local/bin/xray uuid
# =============================================
